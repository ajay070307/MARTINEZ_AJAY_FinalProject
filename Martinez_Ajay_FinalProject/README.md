# Contact Book CLI Application

## Project Description
The **Contact Book** is a Command Line Interface (CLI) application built in Python.  
It allows users to **add, search, update, delete, and list contacts**.  
Contacts are stored persistently in a JSON file, making the application practical for everyday use.

This project demonstrates:
- **Classes/Objects (OOP principles)**  
- **File Handling (read/write JSON)**  
- **String Processing (search, normalization, validation)**  

---

## Features
- Add new contacts (name, phone, email)
- List all saved contacts
- Search contacts by name, phone, or email
- Update existing contact details
- Delete contacts
- Persistent storage using `contacts.json`
- Input validation and error handling
